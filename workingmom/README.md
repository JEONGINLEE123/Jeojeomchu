# 도란도란 집안일

부부가 함께 오늘 필요한 집안일만 확인하고 완료하는 모바일 우선 웹앱입니다.

## 실행

`index.html`을 브라우저에서 직접 열어도 한 기기용으로 동작합니다.

부부의 서로 다른 휴대폰에서 함께 쓰려면 이 폴더에서 다음 명령을 실행하세요.

```powershell
npm install
npm start
```

터미널에 나타나는 `같은 와이파이` 주소를 두 휴대폰에서 똑같이 열면 됩니다. 서버를 실행한 컴퓨터와 휴대폰은 같은 와이파이에 연결되어 있어야 합니다.

## 현재 구현 범위

- 기본 집안일과 반복 주기 계산(빨래는 부름이 / 수건·속옷·흰옷 / 검정옷으로 분리)
- 밀린 일 / 오늘 권장 / 해도 되는 일 자동 분류
- 카드에서 바로 완료하고 필요할 때 완료자 변경, 내가 할게, 맡기 취소, 내일 하자, 다른 날짜로 미루기, 아직 괜찮음
- 완료·내일 하자·장보기 변경을 10초 안에 되돌리기
- 묶음 체크, 조건부 젖병 소독 생성, 함께 완료
- 엄마의 월·화 23:00~08:00 야간근무와 화·수 08:00~18:00 회복 시간, 아빠의 10:30~21:30 근무 반영
- 회복 시간에는 엄마 대상 집안일 추천·푸시 알림을 쉬고, 화요일 저녁에는 밤 근무 전 가벼운 일만 안내
- 아빠의 매주 달라지는 휴무일을 5주 달력에서 지정하고 주간 복사·최근 휴무 패턴 적용
- 근무 중인 사람에게 자동 배정하지 않고 가능한 시간·함께하기 좋은 시간만 안내
- 전체 집안일 활성화 관리 및 직접 간격·특정 요일·세부 항목·잠시 쉬기를 지원하는 사용자 작업 추가
- 월간 미니 달력에서 날짜를 골라 완료자·점검 기록·그날 하지 않은 일 확인, 주간 수고 요약과 반복 미루기 확인
- 날짜별 완료 취소와 그날 기록 초기화, 내일로 넘긴 일 기록
- 설정을 보존하는 `오늘 체크 초기화`와 기록 화면의 날짜별 `이날 체크 초기화`
- 이름, 하루 경계 시각, 업무량 통계 설정
- 휴대폰별 기본 사용자 기억
- `localStorage` 저장, 오프라인 사용 및 재연결 시 자동 동기화
- 항목 단위 병합으로 두 휴대폰의 동시 수정 보호
- 오전 요약·저녁 남은 일·선택적 배우자 활동 웹 푸시 알림
- 하루 한 번 최근 31일 자동 백업, 설정의 날짜별 자동 복원 지점, 백업 내보내기·불러오기
- 가족 로그인 후 이 기기에서 90일간 유지되는 보안 자동 로그인
- Railway 공동 서버와 영구 볼륨을 통한 와이파이·모바일 데이터 접속 및 기록 공유
- 두 사람이 함께 쓰는 장보기 목록(분류별 묶음, 최근 산 품목 재추가, 품목 수정, 장보기 모드, 구매 체크)
- 홈 화면 설치 안내, 새 버전 적용 안내, 공동 데이터 동기화 상태 표시

## `workingmom.jeojeomchu.xyz` 배포

현재 서버는 Railway 배포에 필요한 `PORT` 환경변수를 지원합니다. 별도 Railway 서비스로 배포한 뒤 아래처럼 설정합니다.

1. Railway 서비스의 Public Networking에서 `workingmom.jeojeomchu.xyz`를 Custom Domain으로 추가합니다.
2. Railway가 표시하는 CNAME과 소유권 확인용 TXT 레코드를 가비아 DNS 관리툴에 그대로 추가합니다.
3. 기록 보존용 Railway Volume을 `/data`에 연결하고 환경변수 `DORAN_DATA_DIR=/data`를 설정합니다.
4. 외부 접근 보호를 위해 `WORKINGMOM_USERNAME`과 `WORKINGMOM_PASSWORD` 환경변수를 설정합니다. 로그인 성공 시 비밀번호를 브라우저에 저장하지 않고 `HttpOnly` 세션 쿠키를 발급합니다.
5. 필요하면 `WORKINGMOM_SESSION_SECRET`에 충분히 긴 임의 문자열을 설정해 세션 서명 키를 비밀번호와 별도로 관리할 수 있습니다.

기존 `jeojeomchu.xyz` 서비스와 분리해야 점심추천 앱의 배포나 장애가 WorkingMOM에 영향을 주지 않습니다.
