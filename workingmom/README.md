# 도란도란 집안일

부부가 함께 오늘 필요한 집안일만 확인하고 완료하는 모바일 우선 웹앱입니다.

## 실행

`index.html`을 브라우저에서 직접 열어도 한 기기용으로 동작합니다.

부부의 서로 다른 휴대폰에서 함께 쓰려면 이 폴더에서 다음 명령을 실행하세요.

```powershell
node server.js
```

터미널에 나타나는 `같은 와이파이` 주소를 두 휴대폰에서 똑같이 열면 됩니다. 서버를 실행한 컴퓨터와 휴대폰은 같은 와이파이에 연결되어 있어야 합니다.

## 현재 구현 범위

- 22개 기본 집안일과 반복 주기 계산
- 밀린 일 / 오늘 권장 / 해도 되는 일 자동 분류
- 내가 할게, 맡기 취소, 오늘 미루기, 아직 괜찮음
- 묶음 체크, 조건부 젖병 소독 생성, 함께 완료
- 엄마의 화·수 23:00~08:00 야간근무와 아빠의 10:30~21:30 근무 반영
- 아빠의 매주 달라지는 휴무일을 2주 달력에서 간편하게 지정
- 근무 중인 사람에게 자동 배정하지 않고 가능한 시간·함께하기 좋은 시간만 안내
- 전체 집안일 활성화 관리 및 사용자 작업 추가
- 날짜별 완료 기록과 완료 취소
- 설정을 보존하는 `오늘 체크 초기화`와 기록 화면의 날짜별 `이날 체크 초기화`
- 이름, 하루 경계 시각, 업무량 통계 설정
- `localStorage` 저장 및 같은 브라우저 탭 사이 `BroadcastChannel` 동기화
- 내장 공동 서버를 통한 같은 와이파이 기기 간 실시간 반영

컴퓨터를 계속 켜두지 않고 외부에서도 사용하려면 이후 Firebase, Supabase 같은 인증·클라우드 백엔드로 옮길 수 있습니다.

## `workingmom.jeojeomchu.xyz` 배포

현재 서버는 Railway 배포에 필요한 `PORT` 환경변수를 지원합니다. 별도 Railway 서비스로 배포한 뒤 아래처럼 설정합니다.

1. Railway 서비스의 Public Networking에서 `workingmom.jeojeomchu.xyz`를 Custom Domain으로 추가합니다.
2. Railway가 표시하는 CNAME과 소유권 확인용 TXT 레코드를 가비아 DNS 관리툴에 그대로 추가합니다.
3. 기록 보존용 Railway Volume을 `/data`에 연결하고 환경변수 `DORAN_DATA_DIR=/data`를 설정합니다.
4. 외부 접근 보호를 위해 `WORKINGMOM_USERNAME`과 `WORKINGMOM_PASSWORD` 환경변수를 설정합니다. 비밀번호를 설정하면 브라우저가 가족 로그인을 요청합니다.

기존 `jeojeomchu.xyz` 서비스와 분리해야 점심추천 앱의 배포나 장애가 WorkingMOM에 영향을 주지 않습니다.
